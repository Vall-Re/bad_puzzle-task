# Puzzle Task

Задача
Скласти найбільший однорядний цифровий пазл.

## Як запустити проєкт
Відповідно до Обмеження #1, проєкт максимально доступний і не потребує складної інсталяції:

1. **Клонуйте репозиторій:**
  ```bash
  git clone https://github.com/Vall-Re/bad_puzzle-task
  ```
2. **Запустіть локальний сервер:** Відкрийте папку з проєктом у вашому редакторі (наприклад, VS Code) та запустіть файл index.html через **Live Server**
3. **Отримайте результат:** Натисніть кнопку "Скласти ланцюжок" на головній сторінці
4. **Зміна даних:** Щоб обробити інший набір фрагментів, просто оновіть масив у файлі data.json.js. Програма автоматично перерахує кількість елементів та побудує новий шлях

## Логіка алгоритму:

Для знаходження послідовності використано **жадібний алгоритм (Greedy Algorithm)**, де кожен цифровий фрагмент розглядається як деталь пазла з двома замками:

- Лівий замок: перші дві цифри числа

- Правий замок: останні дві цифри числа

### Як це працює:
**Старт:** Програма бере перший фрагмент як основу майбутнього ланцюжка
**Пошук:** Вона сканує масив у пошуках деталей, чиї "замки" збігаються з краями ланцюжка
**З'єднання:** 
- Якщо знайшли деталь, де початок збігається з кінцем ланцюжка — додаємо її вправо
- Якщо знайшли деталь, де кінець збігається з початком ланцюжка — додаємо її вліво
**Злиття:** При з'єднанні спільні дві цифри ("замки") накладаються одна на одну, утворюючи безшовний цифровий рядок
**Ефективність:** Використання жадібного підходу дозволяє миттєво обробляти масиви з сотень елементів без перевантаження процесора
